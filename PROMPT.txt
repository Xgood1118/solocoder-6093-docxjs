在 renderAsync 输出 HTML 的基础上新增文档导航面板：渲染时扫描所有 heading 段落（outlineLevel 1-9）生成可折叠的目录树，侧边栏固定定位，点击条目滚动到对应位置，大纲随滚动自动高亮当前章节；在 Options 新增大纲面板默认展开层级（默认 3 级）和隐藏导航的开关。在 renderAsync 输出的文档内容上新增搜索和高亮功能：用户按下快捷键打开搜索框输入关键词，匹配项在文档正文中用黄色背景高亮（支持大小写敏感开关），当前匹配项用橙色高亮，快捷键或回车跳转下一个匹配项并滚动到视区中央，搜索框关闭时自动清除所有高亮。新增自定义样式注入机制：renderAsync 的 Options 新增 userStyles 数组字段可传入 CSS 规则对象（selector 和 properties 两个字段），渲染完成后注入到 styleContainer 中，支持优先级排列使自定义样式能覆盖默认文档样式。renderAsync 渲染完毕后新增导出文档内容为纯文本的方法：遍历渲染后的 DOM 提取文本内容按段落保留换行，表格单元格用制表符分隔，配合 outlineLevel 信息在 heading 前面插入对应层级数的 # 标记生成 markdown 大纲结构。
